Мои навыки

На данный момент я обладаю следующими навыками

  • PUG

    Шаблонизатор html, который позволяет сделать верстку компонентной.

  • SCSS

    Препроцессор CSS, который делает возможным разделение стилей для каждого компонента, а также циклы, переменные и т.д

  • JavaScript

    Владею основами JS, а также некоторыми фишками ES6, замыкания, контекст вызова, что такое setTimeout и setInterval - чем они отличаются, и т.д

  • JQuery

    Простая библиотека для работы с DOM элементами на странице

  • Webpack

    Этот бандлер позволяет мне собрать все файлы моего проекта, проставить перфиксы в css, собрать весь js код и минифицировать это все.

  • React js

    Знаю, что такое PROPS, STATE, что такое контейнерная компонента и презентационная, Lazy load, Context API, React.memo и PureComponent, жизненный цикл компоненты

  • Redux

    Что такое STORE, connect, dispatch, action, action creator, redux-thunk, redux-form, как происходит обмен данными между UI и STORE

  • AXIOS

    Библиотека, которая упрощает взаимодействие с REST API
